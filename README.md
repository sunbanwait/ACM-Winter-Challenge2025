# FaceNav

**🏆 2nd Place Winner - ACM Winter Challenge**

FaceNav is a hands-free mouse interface that uses computer vision to allow users to control their cursors entirely with their head and facial features. Our project was designed specifically to assist students with motor impairments and disabilities while navigating Camino (Santa Clara University’s academic platform).
